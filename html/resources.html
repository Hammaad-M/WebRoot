W3Schools TailwindCSS Docs Coolors.co TinyWow MDN Reference

  1. W3Schools
    Has an easy to read documention of HTML tags, JavaScript functions, and CSS properties.
    Very handy for quickly understanding new content, however is not entirely extensive and sometimes too basic.

    Rating: 4/5

  2. TailwindCSS Docs
    Documentation for TailwindCSS, a utility-based CSS framework that offers the customizability of custom CSS, just a whole lot quicker.
    Documentation is as good as it gets, mapping CSS to the utility classes while even explaining the concepts behind the CSS.

    Rating: 5/5

  3. Coolors.co
    Color scheming website that generates complementary color schemes using any color.
    Very useful for getting an idea where to start when coming up with a color scheme.

    Rating 5/5

  4. TinyWow
    Free online file and image processing tools.
    From removing the background of images to upscaling using AI, this site is incredibly useful.

    Rating 5/5

  5. MDN Reference
    The industry grade documentation for HTML, CSS, and JavaScript. Fully extensive yet not the easiest to read.
    There's always something new to learn here.

    Rating 5/5
